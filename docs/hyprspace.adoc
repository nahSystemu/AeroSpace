== Synopsis
[verse]
hyprspace [-h|--help] [-v|--version] <subcommand> [<subcommand-options>...] [<subcommand-arguments>...]

== Description

HyprSpace is an i3-like tiling window manager for macOS

*hyprspace* command line program is used to manipulate HyprSpace and query its state.

See https://nikitabobko.github.io/HyprSpace/commands for available <subcommand> options

See each <subcommand> individual man page for <subcommand-options> and <subcommand-arguments>

== Options

-h, --help:: Print help
